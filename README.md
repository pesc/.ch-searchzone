> Search the .ch zone file for DNS Records

### 🏠 [Homepage](https://searchzone.ch)

## Prerequisites

-   npm >=7.5.0
-   node >=15.8.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
